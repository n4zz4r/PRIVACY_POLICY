# Kebijakan Privasi bro.stok

Terakhir diperbarui: [Tanggal]

## Pendahuluan

bro.stok berkomitmen melindungi privasi Anda. Dokumen ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan aplikasi.

## Data yang Dikumpulkan

### Informasi
- Informasi akun (jika ada modul akun)
- Informasi bisnis: detail produk/varian dan data persediaan
- Riwayat stok dan aktivitas yang Anda lakukan di aplikasi
- Gambar produk (opsional) yang Anda unggah

### Penyimpanan Data
- Seluruh data disimpan secara lokal di perangkat Anda
- Tidak ada data yang dikirim ke server eksternal
- Data sensitif disimpan menggunakan mekanisme penyimpanan aman

## Penggunaan Data
- Menjalankan fungsi utama manajemen stok
- Membuat ekspor (CSV/XLSX) dan analitik yang Anda minta
- Meningkatkan performa dan pengalaman pengguna
- Memberikan dukungan saat Anda membutuhkan bantuan

## Keamanan Data
- Data disimpan lokal di perangkat
- Kami menerapkan langkah wajar untuk menjaga keamanan data
- Kami tidak membagikan data Anda ke pihak ketiga
- Anda memiliki kontrol penuh terhadap data (lihat, ubah, hapus, ekspor)

## Hak Anda
- **Akses**: Melihat seluruh data di aplikasi
- **Perubahan**: Memperbarui/mengubah data kapan saja
- **Hapus**: Menghapus data melalui fitur aplikasi
- **Ekspor**: Mengekspor data ke CSV/Excel

## Retensi Data
- Data disimpan selama Anda menggunakan aplikasi
- Anda dapat menghapus data kapan saja
- Setelah dihapus, data tidak disimpan kembali oleh aplikasi

## Layanan Pihak Ketiga
- Image Picker (foto produk) – tidak mengirim data ke server
- Berbagi/penyimpanan file (ekspor) – proses lokal pada perangkat
- Secure Storage – untuk menyimpan informasi sensitif secara aman (lokal)

## Privasi Anak
Aplikasi tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan data pribadi anak-anak.
